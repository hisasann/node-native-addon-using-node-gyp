# node-gyp を使った native addon の作り方

[nodejs/node-gyp: Node.js native addon build tool](https://github.com/nodejs/node-gyp)


## Installation

```
$ npm i
```

**.gyp** ファイルがあると `npm i` で自動的に node-gyp が実行される。


## Build

```
$ npm run build
```

or

```
$ npm run rebuild
```


## Test

```
$ npm run test
```
